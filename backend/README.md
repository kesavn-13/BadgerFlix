# BadgerFlix Backend

FastAPI backend for BadgerFlix.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Create `.env` file:
```
GEMINI_API_KEY=your_api_key_here
```

3. Run the server:
```bash
python main.py
```

Server runs on `http://localhost:8000`

API documentation: `http://localhost:8000/docs`
