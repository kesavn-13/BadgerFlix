## What Each Episode Contains

When you upload an MP3 file, the AI generates **4-6 episodes** with the following structure:

### Episode Structure:

```json
{
  "id": "unique-episode-id",
  "course_id": "unique-course-id",
  "title": "Episode 1: Introduction to Machine Learning",
  "summary": "This episode introduces the fundamental concepts of machine learning, including supervised and unsupervised learning paradigms, and explores real-world applications.",
  "key_points": [
    "Definition of machine learning",
    "Supervised vs unsupervised learning",
    "Real-world applications and examples"
  ],
  "transcript": "Machine learning is a subset of artificial intelligence that enables systems to learn from data without being explicitly programmed. In this episode, we'll explore..."
}
```

## ðŸ“‹ Detailed Breakdown

### 1. **Episode Title**
- Format: `"Episode X: [Topic Name]"`
- Example: `"Episode 1: Introduction to Machine Learning"`
- Generated by AI to be descriptive and engaging

### 2. **Summary**
- 2-3 sentence overview of the episode
- Explains what the episode covers
- Example: `"This episode introduces the fundamental concepts of machine learning, including supervised and unsupervised learning paradigms, and explores real-world applications."`

### 3. **Key Points** (Array)
- 3-5 bullet points highlighting main concepts
- Extracted from the transcript
- Example:
  ```json
  [
    "Definition of machine learning",
    "Supervised vs unsupervised learning", 
    "Real-world applications and examples",
    "Types of algorithms"
  ]
  ```

### 4. **Transcript Excerpt**
- 200-300 words from the original transcript
- Relevant portion for this specific episode
- Used for AI Tutor context
